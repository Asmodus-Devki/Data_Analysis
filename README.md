# 🇮🇳 India Election Intelligence Dashboard

Election War Room dashboard built using:

- Streamlit
- Plotly
- Pandas
- OpenPyXL

## Features

- Executive Overview
- Party Analytics
- State Intelligence
- Candidate Intelligence
- Constituency Explorer
- Dynamic Filters
- Download Center

## Run

```bash
pip install -r requirements.txt

streamlit run app.py
```

## Dataset

Indian General Elections 2024
563 Constituencies
36 States / UTs
